PROJECT TITLE: 2048
PURPOSE OF PROJECT: 2048 Game 
VERSION or DATE: 1.0
HOW TO START THIS PROJECT: Run Game2048.java in terminal or open and run
Game2048.java in BlueJ
AUTHORS: Jonathan Ke
USER INSTRUCTIONS:

Use arrow keys to move blocks. Game mechanics identical to 2048 game.

Run Terminal2048.java for the same game but from the terminal.
Run Algorithm.java for an algorithmic robot to play 2048.
Run SimulationAlgorithm.java to collect data on 1000 games of 2048 
played by the bot. Graphing and display tool written into the program for
your convenience!
